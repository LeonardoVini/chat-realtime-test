## Quick start

**WARNING**

> This project was developed with NodeJS v12.16.1 and NPM v6.14.2. Older versions can produce errors, but newer versions are fine.

Frontend
 ```bash
 npm i
 npm start
 ```
 
Backend
 ```bash
 npm i
 nodemon server.js
 ```

Tasks                       | Description
----------------------------|---------------------------------------------------------------------------------------
ng serve -o                 | Start the frontend app in development mode (http://localhost:4200/)
npm run lint                | Run the linter (tslint)
npm install                 | Install dependencies
nodemon server.js           | Start backend app in development mode (http://localhost:3333/)
